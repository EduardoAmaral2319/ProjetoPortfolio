# Portal de Portfólio Profissional 🚀

Este projeto é um Portal de Portfólio dinâmico e responsivo, construído com uma arquitetura baseada em micro-serviços (containers). O objetivo principal é demonstrar o domínio do fluxo de desenvolvimento Full Stack, desde a orquestração da infraestrutura até a criação de uma interface de usuário otimizada e comunicação com o back-end via proxy reverso.

## 🛠️ Tecnologias Utilizadas

O ecossistema do projeto foi construído utilizando as seguintes tecnologias:

* **Front-end:** HTML5 Semântico e CSS3 (Design Mobile-First e aderência rigorosa ao Box Model).
* **Back-end:** Python 3.9 com Flask (API REST para processamento do formulário).
* **Infraestrutura e Proxy Reverso:** Nginx (Alpine).
* **Orquestração:** Docker e Docker Compose.

## 📋 Pré-requisitos

Para rodar este projeto na sua máquina, você não precisa instalar o Python ou o Nginx localmente. É necessário apenas ter instalado:

* [Docker Desktop](https://www.docker.com/products/docker-desktop/) (ou Docker Engine)
* [Git](https://git-scm.com/)
